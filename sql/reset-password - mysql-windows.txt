Root Password Reset for MySQL
=============================

Open two terminals:
- CLIENT: MySQL Client (after failed login but correkt path)
- ADMIN: cmd with administrator priveleges

After this, execute these commands in order:
- CLIENT> mysql --version

- ADMIN> net stop mysql

- CLIENT: mysqld --skip-grant-tables --skip-networking --shared-memory

- CLIENT: mysql -u root
    > FLUSH privileges;
    > USE mysql;
    > ALTER USER 'root'@'localhost' IDENTIFIED BY 'root'
    > QUIT;

- ADMIN: net start mysql

- CLIENT: mysql -u root -p
