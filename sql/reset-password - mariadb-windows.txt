Root Password Reset for MariaDB
===============================

Open two terminals:
- CLIENT: MariaDB Client (after failed login but correkt path)
- ADMIN: cmd with administrator priveleges

After this, execute these commands in order:
- CLIENT: mariadb --version

- ADMIN: net stop mariadb

- CLIENT: mysqld --skip-grant-tables --skip-networking --shared-memory

- CLIENT: mariadb -u root
    > FLUSH privileges;
    > USE mysql;
    > ALTER USER 'root'@'localhost' IDENTIFIED BY 'root'
    > QUIT;

- ADMIN: net start mariadb

- CLIENT: mariadb -u root -p
